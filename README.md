# 🎯 Student Focus Tracker

Student Focus Tracker is a simple, lightweight, and responsive web application designed to help students stay focused while studying. It allows users to track their study sessions, monitor productivity, and build consistent study habits using an intuitive interface.

Built using **HTML**, **CSS**, and **JavaScript**, the application stores all user data locally using **Local Storage**, ensuring that your information remains private and is available even after refreshing the page.

## ✨ Features

- 📚 Track daily study sessions
- ⏱️ Focus timer for productive study
- 📊 Monitor study progress
- 💾 Data stored using Local Storage
- 📱 Fully responsive design
- ⚡ Fast and lightweight
- 🎨 Clean and modern user interface
- 🔒 No sign-up or backend required

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/developer-subham/student-focus-tracker.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.

That's it! No installation or dependencies required.
